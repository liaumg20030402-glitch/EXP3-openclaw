## 作业要求
1. 课前完成问卷 1；
2. 安装 OpenClaw；

5. 完成实验后，完成问卷 2。

## 作业提交
- 实验报告需包括 Skills 和聊天记录；
- OpenClaw 的 log 文件作为附件提交；
- 所有材料打包为压缩包后提交。

## 课堂示例
Skills 本质上是将某类任务的专业流程、规则约束与工具调用方式封装成可复用的能力模块，使通用智能体（Agent）能够像调用函数一样自动完成特定任务。与一次性的 Prompt 不同，Skill 不仅包含自然语言指令，还可以组织资源文件、脚本、示例与评估逻辑，从而支持复杂任务的稳定执行、按需加载与跨平台复用。
本实验使用已经预先编写好的化学 Skills 仓库进行演示，仓库托管在 GitHub 中，包含多个面向化学任务自动化的示例 Skill 与工具型 Skill。
